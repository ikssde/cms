Title: getTranslationInterface()
ShortTitle: Opt_Class::getTranslationInterface()

Reference: Opl_Translation_Interface getTranslationInterface()
Visibility: public
Returns: The current translation interface.
VersionSince: 2.0-beta2
SeeAlso:

Returns the current OPL translation interface object assigned to OPT. The returned value is **null**, if no object is set.
